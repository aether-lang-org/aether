std.audit — query the sandbox audit trail.

import std.audit

aether_audit_count extern

aether_audit_count() -> int

Number of checks currently in the ring buffer.

aether_audit_entry_category extern

aether_audit_entry_category(i) -> string

i: int

Field accessors for buffered entry `i` (0-based, oldest first). The category/resource strings are borrowed — copy before the slot is overwritten. The `entry()` wrapper below does that for you.

count fn

count() -> int

How many permission checks are currently buffered.

entry fn

entry(i) -> (_0, _1, _2)

i: int

Buffered check `i` as a tuple (category, resource, allowed). `allowed` is 1 (passed) or 0 (denied); for an out-of-range index the tuple is ("", "", -1). The strings are Aether-owned copies, safe to keep after further audit activity.

clear fn

clear()

Empty the audit buffer. Scope a later query to just the work that follows by clearing first.

denied_count fn

denied_count() -> int

Convenience: how many of the buffered checks were denials.
