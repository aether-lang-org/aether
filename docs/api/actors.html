register fn

register(name, ref)

name: string, ref: actor_ref

Bind `name` → `ref`. Overwrites any prior binding of the same name (no error). The name is duplicated internally; the actor_ref is stored as-is.

whereis fn

whereis(name) -> actor_ref

name: string

Look up by name. Returns the registered actor_ref, or null (when cast to actor_ref) if no binding exists. Pattern: spawn at startup, register with a stable name, look up from any context (including from inside C-callback handlers that can't take an Aether-typed parameter).

unregister fn

unregister(name) -> int

name: string

Remove a binding. Returns 1 if a binding was removed, 0 if `name` wasn't registered.

is_registered fn

is_registered(name) -> int

name: string

1 if `name` is bound, 0 otherwise.

registry_size fn

registry_size() -> int

Number of currently-registered names. Mostly for tests / debug.

registry_clear fn

registry_clear()

Wipe all bindings. Tests use this for isolation.
