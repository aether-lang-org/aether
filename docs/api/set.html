set

std.set - unordered set of unique strings Backed by the same hash table as std.map, so lookups are O(1) on average and items are copied on insert (the caller's string lifetime does not matter).

free fn

free(set)

set: ptr

Release the set. Items were copied in, so nothing else needs freeing.

items fn

items(set) -> (ptr, string)

set: ptr

Snapshot the set's items, in unspecified order. Returns (MapKeys ptr, "") on success, (null, error) otherwise. The snapshot is released with `set.items_free`.

items_free fn

Release a snapshot returned by `set.items`.
