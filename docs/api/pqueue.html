pqueue

std.pqueue - priority queue over (priority, item) pairs Backed by a binary heap: push and pop are O(log n), peek and size are O(1).

new fn

new() -> ptr

Create an empty queue. Returns null on allocation failure.

push fn

push(pq, priority, item) -> bool

pq: ptr, priority: long, item: ptr

Enqueue `item` at `priority`. Lower priority values pop first. Returns false on a null queue or allocation failure.

pop fn

pop(pq) -> ptr

pq: ptr

Remove and return the lowest-priority item, or null when empty.

peek fn

peek(pq) -> ptr

pq: ptr

The lowest-priority item without removing it, or null when empty.

peek_priority fn

peek_priority(pq) -> long

pq: ptr

Priority of the item `peek` would return. Meaningless when the queue is empty, so check `size` first.

size fn

size(pq) -> int

pq: ptr

Number of queued entries.

is_empty fn

is_empty(pq) -> bool

pq: ptr

True when nothing is queued.

clear fn

clear(pq)

pq: ptr

Drop every entry, keeping the queue usable. Items are not freed.

free fn

free(pq)

pq: ptr

Release the queue. Items were never owned by it, so free them yourself if they were heap allocated.
