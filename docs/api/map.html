put_string_owned extern

put_string_owned(map, key, value) -> int

Heap-string-aware put (#467). Retains the value and tags the map so map_free releases each string value before freeing. Codegen auto-routes `map.put(m, k, heap_string_expr)` here.

put fn

put(map, key, value) -> string

map: ptr, key: string, value: ptr

Insert or update a key-value pair. Returns "" on success, error on failure (null map/key or OOM).

get fn

get(map, key) -> (ptr, string)

map: ptr, key: string

Look up a key. - (value, "") : key present - (null, "") : key absent (not an error — use map.has) - (null, "null map") : `map` is null

keys fn

keys(map) -> (ptr, string)

map: ptr

Snapshot a map's keys. Returns (MapKeys ptr, "") on success, (null, error) on allocation failure. Caller must free with map.keys_free.
