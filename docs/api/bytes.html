aether_bytes_copy_from_bytes extern

aether_bytes_copy_from_bytes(dst, dst_off, src, src_off, length) -> int

dst: ptr, dst_off: int, src: ptr, src_off: int, length: int

Two-buffer copy. `dst` and `src` are distinct AetherBytes pointers; `length` is the count of bytes to copy from `src` at offset `src_off` into `dst` at offset `dst_off`. Grows `dst` if needed. For an in-place copy (same buffer) use `copy_within` — it has the deliberate RLE forward-overlap semantics that memmove() lacks.

`length` here is the count of bytes to copy, not a buffer length.

aether_bytes_finish extern

aether_bytes_finish(b, length) -> string

b: ptr, length: int

`aether_bytes_finish` returns a freshly heap-owned AetherString (`string_new_with_length` — see std/bytes/aether_bytes.c). The `string @heap` return tells the heap-string tracker the caller owns the result, so `x = bytes.finish(...)` gets a `_heap_x = 1` slot and any function returning a `bytes.finish` value is itself classified heap-returning. Mirrors `aether_strbuilder_finish`.

new fn

new(initial_capacity) -> ptr

initial_capacity: int

---- Aether-side wrappers ---- Allocate a new mutable byte buffer with at least `initial_capacity` bytes reserved. Returns null on allocation failure or negative capacity. Initial logical length is 0.

set fn

set(b, index, value) -> int

b: ptr, index: int, value: int

Write a single byte at `index`. The buffer grows if needed; gaps between the previous tail and `index` are zero-filled. Returns 1 on success, 0 on failure. The parameter is named `value` (not `byte`) because `byte` became a reserved keyword for the new primitive type. The third arg's runtime semantics are unchanged — it stores the low 8 bits at `index`.

get fn

get(b, index) -> int

b: ptr, index: int

Read the byte previously written at `index`, returned as 0..255. Returns -1 if `b` is null, `index` is negative, or `index` is past the buffer's logical length. Pairs with `set` so a caller can build then walk a buffer in-place — the canonical binary-codec encoder pattern (svndiff and similar).

length fn

length(b) -> int

b: ptr

Number of bytes the buffer logically contains. Returns -1 if `b` is null.

set_le16 fn

set_le16(b, index, value) -> int

b: ptr, index: int, value: int

Little-endian 16-bit write at `index..index+1`. Grows the buffer if needed. Returns 1 on success, 0 on failure (NULL buffer or negative index).

get_le16 fn

get_le16(b, index) -> int

b: ptr, index: int

Little-endian 16-bit read at `index..index+1`. Returns the unsigned 16-bit value as an int, or -1 on failure (NULL buffer, negative index, or range past current length).

set_le32 fn

set_le32(b, index, value) -> int

b: ptr, index: int, value: int

Little-endian 32-bit write at `index..index+3`. Grows the buffer if needed. Returns 1 on success, 0 on failure.

get_le32 fn

get_le32(b, index) -> int

b: ptr, index: int

Little-endian 32-bit read at `index..index+3`. Returns the value as an int (host-endian after reassembly), or -1 on failure. Round-trips losslessly with set_le32 for any int input.

copy_from_string fn

copy_from_string(b, dst, src, src_len) -> int

b: ptr, dst: int, src: string, src_len: int

Copy `src_len` bytes from `src` into the buffer starting at offset `dst`. `src` may be either a plain `string` literal or an AetherString-bearing value (binary-safe via aether_string_data on the C side). Returns 1 on success, 0 on failure.

copy_from_bytes fn

copy_from_bytes(dst, dst_off, src, src_off, length) -> int

dst: ptr, dst_off: int, src: ptr, src_off: int, length: int

Copy `length` bytes from `src` buffer at offset `src_off` into `dst` buffer at offset `dst_off`. The buffers must be distinct; for an in-place copy use `copy_within`. Grows the destination buffer if needed. Returns 1 on success, 0 on failure (NULL buffer, negative offsets/length, source range past src's logical length, or destination grow OOM). Canonical use: two-pass separable Gaussian blur (pixels → temp → pixels).

copy_within fn

copy_within(b, dst, src, length) -> int

b: ptr, dst: int, src: int, length: int

Copy `length` bytes from offset `src` to offset `dst` *within the same buffer*, forward byte-by-byte. Bytes written earlier in the same call are visible to later reads — the RLE-overlap behaviour. Returns 1 on success, 0 on failure (NULL buffer, negative offsets, or src + length > current buffer length).

finish fn

finish(b, length) -> string

b: ptr, length: int

Hand the buffer off to a refcounted AetherString and destroy the AetherBytes wrapper. After this call, `b` is invalid. The string carries the explicit length so embedded NULs survive end-to-end. Pass the prefix you actually want to keep (use bytes.length(b) to take the whole buffer).

free fn

free(b)

b: ptr

Discard the buffer without converting to a string. Idempotent on null. Use this for error paths that mid-way decide the buffer isn't needed.
