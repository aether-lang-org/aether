nanoid

std.nanoid — NanoID (https://github.com/ai/nanoid).

import std.nanoid

generate fn

generate()

Generate a 21-char NanoID. Returns (id, "") on success, ("", error) on CSPRNG failure.

generate_n fn

generate_n(n) -> (string, string)

n: int

Generate an `n`-char NanoID with the standard URL-safe alphabet. `n` must be >= 1. For collision math: ~2.7 million IDs/sec for ~1000 years to have a 1% collision probability at n=21. Cut `n` only if you understand the birthday-bound implication.
