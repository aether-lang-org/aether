deque

std.deque — a fixed-capacity double-ended queue / ring buffer of `long` values (a `long` holds any int, and a pointer via mem.ptr_to_long, so this backs work queues, sliding windows, and FIFO/LIFO buffers uniformly).

import std.deque

Deque struct

struct Deque

A ring buffer. `head` is the index of the current front element; `count` is how many are live. The backing store holds `capacity` slots.

new fn

new(capacity) -> Deque

capacity: int

Create a deque holding up to `capacity` elements (capacity >= 1).

free fn

free(d)

d: Deque

Release the backing store. The Deque value is invalid afterwards.

push_back fn

push_back(d, value) -> Deque

d: Deque, value: long

Append at the back. On a full buffer this OVERWRITES the oldest (front) element and advances the front — the ring-buffer/sliding-window behaviour. Returns the updated Deque (value semantics: reassign it, `d = deque.push_back(d, x)`).

push_front fn

push_front(d, value) -> Deque

d: Deque, value: long

Prepend at the front. On a full buffer this OVERWRITES the newest (back) element. Returns the updated Deque.

pop_front fn

pop_front(d) -> (long, Deque, string)

d: Deque

Remove and return the front element as (value, deque, ""), or (0, deque, error) when empty. Reassign the returned deque.

pop_back fn

pop_back(d) -> (long, Deque, string)

d: Deque

Remove and return the back element as (value, deque, ""), or (0, deque, error) when empty.

peek_front fn

peek_front(d) -> (long, string)

d: Deque

Read the front element without removing it: (value, "") or (0, error).

peek_back fn

peek_back(d) -> (long, string)

d: Deque

Read the back element without removing it.

clear fn

clear(d) -> Deque

d: Deque

Empty the deque (keeps capacity). Returns the updated Deque.
