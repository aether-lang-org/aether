audio

std.audio — audio playback (v1: playback tier).

import std.audio

open fn

open() -> bool

Initialise the audio engine. Call once before loading/playing. Returns true on success. v1 always succeeds (null backend).

close fn

close()

Shut the engine down. Unload sounds first.

is_null_backend fn

is_null_backend() -> bool

True when the active backend is the NULL (silent) backend — the v1 default. Lets a caller / test assert deterministic, device-free behaviour.

load_wav fn

load_wav(data, length) -> ptr!

data: string, length: int

Decode `length` bytes of `data` (a WAV byte string — e.g. from fs.read_binary) into a playable source. Returns the source on success, or a non-empty error on malformed / unsupported input. The source is owned by the caller: release it with `unload`.

last_error fn

last_error() -> string

The reason the most recent load failed (or "" after a success). BORROWED from the substrate — a static C string valid until the next load_wav.

play fn

play(src) -> bool

src: ptr

Start or resume playback from the current position. Returns true on success (false only for a null source). A source that had reached its end rewinds to the start.

pause fn

pause(src) -> bool

src: ptr

Halt playback, keeping the current position for a later `play`.

stop fn

stop(src) -> bool

src: ptr

Halt playback and rewind to the start.

is_playing fn

is_playing(src) -> bool

src: ptr

True while the source is actively playing (false when paused, stopped, or finished).

volume fn

volume(src, v) -> bool

src: ptr, v: float

Set the source's volume in [0.0, 1.0] (clamped). Returns true on success.

get_volume fn

get_volume(src) -> float

src: ptr

The source's current volume in [0.0, 1.0].

seek_ms fn

seek_ms(src, ms) -> bool

src: ptr, ms: long

Seek to `ms` milliseconds from the start (clamped to [0, duration]). Returns true on success. Keeps the play/pause state.

position_ms fn

position_ms(src) -> long

src: ptr

The current play position in milliseconds — poll this from a UI tick to drive a seek slider. (Milliseconds rather than Duration because the language has no Duration-from-ns constructor yet; a Duration accessor is a clean later addition.)

duration_ms fn

duration_ms(src) -> long

src: ptr

The total length of the source in milliseconds.

channels fn

channels(src) -> int

src: ptr

sample_rate fn

sample_rate(src) -> int

src: ptr

unload fn

unload(src)

src: ptr

Release a source. After this the pointer is invalid.
