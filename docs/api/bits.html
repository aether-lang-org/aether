import std.bits

lsr32 fn

lsr32(x, n) -> int

x: int, n: int

Logical (unsigned) right shift of `x` by `n` bits. The shift count is masked by (W-1), so n is taken mod 32 (lsr32) / mod 64 (lsr64).

rotr32(x, n) -> int

x: int, n: int

Bit rotates. Count masked by (W-1); a count of 0 returns x unchanged.

clz32(x) -> int
