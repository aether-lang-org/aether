longarr

Copyright (c) 2026 Aether Developers.

import std.longarr

new fn

new(size) -> (ptr, string)

size: int

Allocate a LongArray of `size` elements, zero-initialised. Returns (ptr, "") on success, (null, error) on failure.

new_filled fn

new_filled(size, init) -> (ptr, string)

size: int, init: long

Allocate and fill with `init` at every index.

get fn

get(arr, i) -> (long, string)

arr: ptr, i: int

Bounds-checked read. Returns (value, "") on success, (0, error) for null array or OOB index. Use the raw `longarr_get_raw` directly (imported above) when you don't need to tell absent apart from stored-zero — it's one fewer function call and one fewer branch.

set fn

set(arr, i, value) -> string

arr: ptr, i: int, value: long

Bounds-checked write. Returns "" on success, error for null array or OOB index. Use the raw `longarr_set_raw` directly in hot paths where a no-op-on-OOB is acceptable.

new_raw extern

new_raw(size) -> ptr

size: int

Allocate a LongArray of `size` longs, zero-initialised. Returns null if `size` is negative or allocation fails.

new_filled_raw extern

new_filled_raw(size, init) -> ptr

size: int, init: long

Allocate and fill every index with `init`.

size extern

size(arr) -> int

arr: ptr

Number of elements. -1 if `arr` is null.

get_raw extern

get_raw(arr, i) -> long

arr: ptr, i: int

Read at `i`. Returns 0 if `arr` is null or `i` is out-of-range. Use `longarr.get` (below) to distinguish stored-zero from OOB.

set_raw extern

set_raw(arr, i, value)

arr: ptr, i: int, value: long

Write `value` at `i`. No-op if `arr` is null or `i` is out-of-range. Use `longarr.set` (below) if you need an error on OOB.

get_unchecked extern

get_unchecked(arr, i) -> long

arr: ptr, i: int

Hot-path variants — caller guarantees index is in [0, size). OOB access is undefined behaviour. Don't use from outside a tight loop.

set_unchecked extern

set_unchecked(arr, i, value)

arr: ptr, i: int, value: long

fill extern

fill(arr, value)

arr: ptr, value: long

Fill every element with `value`.

free extern

free(arr)

arr: ptr

Release. Idempotent on null.
