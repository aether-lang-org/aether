finish fn

finish(b) -> string

b: ptr

Finish the document and return it as an owned string (escaping already applied throughout). The builder is still alive — call `free_builder`. The string_concat copies the C buffer into an owned Aether string and lets the codegen reclaim the malloc'd original at scope exit (the same handoff std.json's `stringify` uses).
