time

std.time — civil date/time over Unix epoch seconds (UTC).

import std.time

DateTime struct

struct DateTime

A civil UTC date/time. `epoch` (epoch seconds) is canonical; the civil fields are a decoded view of it. Build with from_civil / from_unix / now; read fields directly.

to_iso8601 fn

to_iso8601(dt) -> string

dt: DateTime

Format as "YYYY-MM-DDTHH:MM:SSZ" (RFC-3339 UTC). Years are zero-padded to at least 4 digits.

parse_iso8601 fn

parse_iso8601(s) -> (DateTime, string)

s: string

Parse "YYYY-MM-DDTHH:MM:SS" with an optional trailing 'Z' (UTC). A space may replace the 'T'. Returns the DateTime, or a non-empty error on a malformed shape / out-of-range field.
