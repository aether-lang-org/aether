int_search fn

int_search(arr, x) -> int

arr: ptr, x: int

Binary search a SORTED intarr for `x`. Returns the index of `x` if present, otherwise the insertion point (the index where `x` would be inserted to keep the array sorted) — matching C3's binarysearch contract.
