casper

std.casper — FreeBSD Casper service delegation.

import std.casper

aether_casper_init extern

aether_casper_init() -> ptr

Connect to the casper daemon. Returns an opaque channel, or null. Must be called BEFORE capsicum.enter().

aether_casper_service_open extern

aether_casper_service_open(casper, service) -> ptr

casper: ptr, service: string

Open a named service channel ("system.net" / "system.pwd" / "system.sysctl") from the casper handle. Returns a channel or null. Must be called BEFORE capsicum.enter().

aether_casper_close extern

aether_casper_close(chan)

chan: ptr

Close a channel (casper handle or service handle). Safe on null.

aether_casper_available extern

aether_casper_available() -> int

1 if Casper is usable (FreeBSD with the casper daemon reachable).

aether_casper_resolve extern

aether_casper_resolve(netchan, hostname) -> string

netchan: ptr, hostname: string

Resolve a hostname to its first IP via a "system.net" channel. Returns the address string, or null on failure.

aether_casper_pwd_uid extern

aether_casper_pwd_uid(pwdchan, username) -> int

pwdchan: ptr, username: string

Look up a username's uid via a "system.pwd" channel. -1 if absent.

aether_casper_pwd_home extern

aether_casper_pwd_home(pwdchan, username) -> string

pwdchan: ptr, username: string

Look up a username's home directory via a "system.pwd" channel. Returns the path, or null on failure.

aether_casper_sysctl_str extern

aether_casper_sysctl_str(sysctlchan, name) -> string

sysctlchan: ptr, name: string

Read a string-valued sysctl via a "system.sysctl" channel. Returns the value, or null on failure.
