add_string_owned extern

add_string_owned(list, item) -> int

list: ptr, item: ptr

Heap-string-aware add (#467). Retains the value and tags the list so list_free releases each string element before freeing the backing array. Codegen auto-routes `list.add(l, heap_string_expr)` here when the value is heap-classified.

Index into a list. - (item, "") : index in range - (null, "") : index out of range (not an error) - (null, "null list") : `list` is null
