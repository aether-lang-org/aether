put fn

put(key, value)

key: string, value: string

Insert / overwrite. Both `key` and `value` are duplicated internally; the caller's string lifetimes don't matter.

get fn

get(key) -> string

key: string

Retrieve. Returns "" if the key isn't set (matches Aether's Go-style "" = absent convention).

get_or fn

get_or(key, default_value) -> string

key: string, default_value: string

Retrieve with a fallback. Returns the registered value if set, otherwise `default_value`.

has fn

has(key) -> int

key: string

1 if `key` has been put, 0 otherwise.

size fn

size() -> int

Number of keys currently registered. Mostly for tests / debug.

clear fn

clear()

Wipe all keys. Tests use this for isolation; production code almost never needs it.
