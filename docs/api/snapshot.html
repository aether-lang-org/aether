snapshot

std.snapshot — copy-on-write snapshot cell for read-mostly shared data.

import std.snapshot

aether_snapshot_new extern

aether_snapshot_new(initial) -> ptr

initial: ptr

Create a cell pointing at `initial` (may be null). Returns the cell handle, or null on allocation failure / memory-cap exceeded. Does NOT take ownership of `initial`.

aether_snapshot_load extern

aether_snapshot_load(cell) -> ptr

cell: ptr

Lock-free read: atomic acquire load of the current snapshot. Returns null for a null cell.

aether_snapshot_store extern

aether_snapshot_store(cell, new_value) -> ptr

cell: ptr, new_value: ptr

Publish `new_value` and return the displaced (old) snapshot. The writer now owns the returned pointer and must reclaim it only after a grace period (see the contract above). Null cell returns null.

aether_snapshot_cas extern

aether_snapshot_cas(cell, expected, new_value) -> int

cell: ptr, expected: ptr, new_value: ptr

Compare-and-swap. If the cell holds `expected`, swap in `new_value` and return 1; otherwise return 0 and leave the cell unchanged. On success the caller owns `expected` (reclaim after a grace period); on failure re-load and retry. Null cell returns 0.

aether_snapshot_free extern

aether_snapshot_free(cell)

cell: ptr

Free the cell itself. Does NOT free the snapshot it points at — that value is caller-owned. Null-safe.

cas fn

cas(cell, expected, new_value) -> int

cell: ptr, expected: ptr, new_value: ptr

snapshot.cas(cell, expected, new_value) — read-modify-write CAS; returns 1 on success, 0 on failure. Loop pattern (writer-actor): loop { cur = snapshot.load(cell) next = build_next_from(cur) // pure, allocates `next` if snapshot.cas(cell, cur, next) == 1 { // `cur` is now displaced — reclaim it after a grace // period (e.g. defer one generation), NOT immediately. break } // lost the race: free our unused `next` and retry free_value(next) }

cell: ptr

snapshot.free(cell) — free the cell. Does NOT free the snapshots.
