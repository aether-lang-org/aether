encoding

std.encoding — general-purpose binary/text encodings.

import std.encoding

hex_encode fn

hex_encode(data, length) -> string

data: string, length: int

Encode `length` bytes of `data` as lowercase hex. Output is 2*length characters; empty input yields "".

hex_decode fn

hex_decode(s) -> string!

s: string

Decode a hex string to its bytes. The input must have even length and contain only hex digits. Returns the decoded bytes on success, a non-empty error otherwise.

base64_encode fn

base64_encode(data, length) -> string

data: string, length: int

Standard-alphabet base64 WITHOUT padding (RFC 4648 §4, the `+/` alphabet; no trailing `=`). Encodes `length` bytes of `data`.

base64_encode_padded fn

base64_encode_padded(data, length) -> string

data: string, length: int

Standard-alphabet base64 WITH `=` padding to a multiple of 4 — the form some auth headers and JSON blob formats require. (Standard `+/` alphabet; NOT the url-safe `-_` alphabet.)

base64_decode fn

base64_decode(b64) -> string!

b64: string

Decode a base64 string (accepts padded and unpadded standard-alphabet input) to its bytes. Returns the decoded bytes on success, a non-empty error on malformed input.

base32_encode fn

base32_encode(data, length) -> string

data: string, length: int

Encode `length` bytes of `data` as standard, `=`-padded base32.

base32_decode fn

base32_decode(s) -> string!

s: string

Decode a standard-alphabet base32 string (with or without `=` padding) to its bytes. Returns the decoded bytes, or a non-empty error on a character outside the alphabet.

csv_split fn

csv_split(record, sep) -> ptr

record: string, sep: string

Split one CSV record into fields on `sep`. Returns an owned handle; read it with csv_count / csv_field, then release with csv_free. A trailing "\r" (from a "\r\n" line ending) is trimmed before splitting.

csv_count fn

csv_count(handle) -> int

handle: ptr

Number of fields in a handle returned by csv_split.

csv_field fn

csv_field(handle, i) -> string

handle: ptr, i: int

Field `i` (0-based) as a string. Out-of-range yields "". The returned string is BORROWED from the handle — valid until csv_free(handle); copy it (string.concat(f, "")) if you need to hold it past the free.

csv_free fn

csv_free(handle)

handle: ptr

Release a handle returned by csv_split.
