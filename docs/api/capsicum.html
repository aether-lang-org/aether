capsicum

std.capsicum — FreeBSD Capsicum capability-mode bindings.

import std.capsicum

CAP_OK const

CAP_OK = 0

CAP_ERR const

CAP_ERR = -1

CAP_UNSUPPORTED const

CAP_UNSUPPORTED = -2

aether_capsicum_available extern

aether_capsicum_available() -> int

1 if Capsicum is usable in this process, 0 otherwise.

aether_capsicum_enter extern

aether_capsicum_enter() -> int

Enter capability mode. Irreversible. Returns CAP_OK / CAP_ERR / CAP_UNSUPPORTED.

aether_capsicum_in_mode extern

aether_capsicum_in_mode() -> int

1 if already in capability mode, 0 if not, CAP_UNSUPPORTED off FreeBSD.

aether_capsicum_rights_limit extern

aether_capsicum_rights_limit(fd, rights_mask) -> int

fd: int, rights_mask: int

Limit `fd` to the rights in `rights_mask` (OR of R_* values). Rights only ever narrow. Returns CAP_OK / CAP_ERR / CAP_UNSUPPORTED.

aether_capsicum_fcntls_limit extern

aether_capsicum_fcntls_limit(fd, fcntl_mask) -> int

fd: int, fcntl_mask: int

Limit which fcntl(2) commands `fd` permits (OR of F_* values).
