signal

std.signal — POSIX signal-number constants.

import std.signal

SIGHUP fn

SIGHUP() -> int

Hangup — controlling terminal closed / daemon reload convention.

SIGINT fn

SIGINT() -> int

Interrupt — what Ctrl-C raises. Forwarded to the build group by os.run_supervised when forward_signals is on.

SIGQUIT fn

SIGQUIT() -> int

Quit — Ctrl-\, like SIGINT but with a core dump.

SIGILL fn

SIGILL() -> int

Illegal instruction.

SIGABRT fn

SIGABRT() -> int

Abort — what abort(3) / assert(3) raise.

SIGFPE fn

SIGFPE() -> int

Floating-point exception (also integer divide-by-zero).

SIGKILL fn

SIGKILL() -> int

Kill — cannot be caught, blocked, or ignored. The "grace expired, take it down now" signal os.run_supervised sends after SIGTERM.

SIGSEGV fn

SIGSEGV() -> int

Invalid memory reference.

SIGPIPE fn

SIGPIPE() -> int

Write to a pipe with no reader.

SIGALRM fn

SIGALRM() -> int

Timer signal from alarm(2).

SIGTERM fn

SIGTERM() -> int

Termination — the polite "please shut down" request. The default signal os.run_supervised sends first on timeout / group teardown.
